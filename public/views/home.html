Specialties

User Interfaces - Designed to be easy and enjoyable for users.

  • Sleek single page web sites and applications mean less load times for your users
  • Approachable designs based on Twitter Bootstrap, with the ability to create custom "themes" for your web site or application
  • Responsive and cross-platform design means your users can view your web site or application on any browser on any device

User Experience - Tailored to turn your project into a cohesive experience for users.

  • Focused on simple design to not "make the user think", and instead jump right into your web site or application
  • Design that allows the user to explore your web page or application, or navigate to directly where they need to go in as few clicks as possible
  • Use intuitive color schemes so users can understand buttons, alerts and notifications at glance

Team Development - Accustomed to work with designers, engineers and user/testers.

  • Implement designs and features according to specification
  • Quick and iterative development using project management systems like AGILE
  • Well maintained version control practices using technology like Git

Communication - Written and verbal communication skills help to schedule, design and develop around your requirements.

  • Capable of working remote using technologies like WebEx and to share ideas
  • Understand conveying information in a variety of mediums
  • Respond promptly and professionally, never leaving you in the dark
